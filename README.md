# Polza Outreach Engine — Тестовое задание

## 1. Проверка email-доменов

``` bash
pip install dnspython 
python check_domains.py
```

Скрипт проверяет список email-адресов и выводит один из трёх статусов:
- домен валиден
- домен отсутствует
- MX-записи отсутствуют или некорректны

Тестовые адреса уже есть в коде. При необходимости просто замените список test_emails.

## 2. Отправка сообщения в приватный Telegram-чат

Подготовка (один раз):
1. Создайте бота через @BotFather → получите токен
2. Вставьте токен в send_to_telegram.py вместо текущего значения BOT_TOKEN
3. Положите нужный текст в message.txt

``` bash
pip install requests
python send_to_telegram.py
```

При первом запуске: напишите боту любое сообщение → скрипт сам заберёт chat_id и сохранит его в chat_id.txt.
Дальше отправка происходит мгновенно.
